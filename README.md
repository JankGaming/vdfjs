# VDF.js

Convert Valve's KeyValue format to JSON

## Install

`npm install vdfjs`

## Usage

### In a script

```javascript
var vdf = require('vdfjs');

var jsonString = vdf.parse(someVDFfile);
```

### CLI
`node vdf.js path_to_file`
